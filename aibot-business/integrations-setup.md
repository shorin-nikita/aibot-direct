# 🏢 Настройка бизнес-интеграций

## 🚀 Быстрый старт

### 1. Импорт workflow'ов в n8n
```bash
# После запуска AIBot Direct:
# 1. Перейти в http://localhost:5678
# 2. Settings → Import from file
# 3. Выбрать файлы из n8n-business-workflows/
```

### 2. Telegram бот
```bash
# Создание бота:
# 1. Написать @BotFather в Telegram
# 2. /newbot → выбрать имя → получить токен
# 3. Добавить токен в переменную TELEGRAM_BOT_TOKEN в workflow
```

## 📈 Битрикс24

### Настройка
1. **Получить webhook URL**:
   - Битрикс24 → Приложения → Вебхуки
   - Создать входящий вебхук
   - Скопировать URL

2. **Настроить права**:
   - CRM (чтение/запись)
   - Задачи (чтение/запись) 
   - Пользователи (чтение)

3. **В n8n workflow**:
   - Заменить `yourdomain.bitrix24.ru` на ваш домен
   - Вставить webhook ключи

### Возможности
- ✅ Автоматический анализ новых лидов
- ✅ AI оценка качества лидов
- ✅ Уведомления в Telegram о горячих лидах
- ✅ Автоматические комментарии с рекомендациями

## 💼 AmoCRM

### Настройка
1. **API ключи**:
   - AmoCRM → Настройки → API и веб-хуки
   - Создать интеграцию
   - Получить CLIENT_ID и CLIENT_SECRET

2. **OAuth авторизация**:
   - Настроить redirect_uri
   - Получить access_token

3. **В n8n workflow**:
   - Заменить `yourdomain.amocrm.ru`
   - Добавить токены

### Возможности
- ✅ Анализ закрытых сделок
- ✅ AI рекомендации по работе с клиентами
- ✅ Автоматические заметки в карточках
- ✅ Прогнозирование успешности сделок

## 📱 Telegram голосовые заказы

### Настройка ресторана/доставки
1. **Создать канал персонала**:
   - Создать закрытый канал для уведомлений
   - Получить chat_id канала
   - Добавить бота в администраторы

2. **В workflow**:
   - Заменить `@restaurant_orders` на ваш канал
   - Настроить меню и цены в AI промпте

### Возможности
- ✅ Приём заказов голосом через Telegram
- ✅ Автоматическое распознавание блюд
- ✅ Расчёт стоимости заказа
- ✅ Уведомления персонала в реальном времени

## 🔧 Общие настройки

### Переменные окружения
```bash
# Добавить в .env файл:
TELEGRAM_BOT_TOKEN=your_bot_token
BITRIX24_WEBHOOK_KEY=your_webhook_key
AMOCRM_ACCESS_TOKEN=your_access_token
GIGACHAT_CLIENT_ID=your_gigachat_id
YANDEX_CLOUD_API_KEY=your_yandex_key
```

### Безопасность
- 🔒 Все API ключи хранятся локально
- 🔒 Данные не передаются третьим лицам
- 🔒 Полный контроль над информацией
- 🔒 Соответствие требованиям по хранению данных РФ
