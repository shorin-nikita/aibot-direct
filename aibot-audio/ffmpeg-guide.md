## 🔧 Технические детали

### Whisper API
```
Endpoint: http://localhost:9000/asr
Method: POST
Content-Type: multipart/form-data

Parameters:
- audio_file: файл для распознавания
- language: "ru" (русский язык)
- model: "base" (базовая модель)
```

### Поддерживаемые форматы
```
Аудио: mp3, wav, ogg, m4a, flac, aac
Видео: mp4, avi, mov, mkv, webm
Максимальный размер: 25MB (можно увеличить)
```

### FFmpeg команды

#### Конвертация для Whisper
```bash
# MP3 в WAV (оптимально для распознавания)
ffmpeg -i input.mp3 -ar 16000 -ac 1 output.wav

# Видео в аудио для транскрипции
ffmpeg -i meeting.mp4 -vn -acodec pcm_s16le output.wav

# Сжатие большого файла
ffmpeg -i large_file.wav -ab 64k -ar 16000 -ac 1 compressed.wav
```

#### Обработка звонков
```bash
# Разделение стерео на 2 канала (клиент/менеджер)
ffmpeg -i call.wav -map_channel 0.0.0 client.wav -map_channel 0.0.1 manager.wav

# Улучшение качества голоса
ffmpeg -i input.wav -af "highpass=f=200,lowpass=f=3000" clean_voice.wav
```
